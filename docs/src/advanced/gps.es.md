# Editar con trazas GPS

En la sección [CoMaps](../apps/comaps.md) mostramos cómo grabar una traza GPS.
Las trazas GPS también se pueden grabar de muchas otras maneras. Tradicionalmente, 
se utilizaban receptores GPS, como los de la marca Garmin, pero hoy en día las aplicaciones para smartphones 
o incluso los relojes GPS permiten hacer lo mismo.

El objetivo principal es guardar la traza GPS en un archivo GPX. Un archivo GPX es un formato 
universal para intercambiar información de trazas o rutas GPS entre aplicaciones geográficas. 
En [OpenStreetMap.org](https://openstreetmap.org) puedes subir archivos GPX y usarlos para dibujar nuevas carreteras, rutas y caminos.

Los archivos GPX también se usan para compartir rutas de senderismo, como en AllTrails o WikiLoc. 
Los archivos GPX se pueden abrir, por ejemplo, en CoMaps y usar para seguir una ruta. 
En este caso, usamos nuestros propios registros GPS.

## Videotutorial

Acabo de crear un videotutorial sobre cómo:

* Iniciar sesión en [OpenStreetMap.org](https://openstreetmap.org) (asegúrese de crear primero una cuenta de OSM)
* Subir su traza GPS como archivo GPX
* Editar el mapa usando la traza GPS como fondo de pantalla

El tutorial presupone que:

* Conoce los conceptos básicos del [Editor de ID de OSM](https://learnosm.org/en/beginner/id-editor/)
* Ha grabado una traza GPS con, por ejemplo, [CoMaps](../apps/comaps.md)
